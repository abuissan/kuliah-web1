Penyu

Penyu di Indonesia sekarang terancam punah dan semakin sulit ditemui di laut. Penyebab dari langkanya satwa air ini karena banyaknya pemangsa yang mengincar tukik atau anak penyu dan perburuan telur penyu yang membuat populasi penyu semakin lama semakin berkurang.

Harimau Sumatera

Harimau Sumatera merupakan salah satu hewan yang dilindungi oleh pemerintah Indonesia. Beberapa penyebab hewan ini menjadi langka adalah penebangan hutan yang sembarangan, dan perburuan liar.

Komodo

Komodo merupakan hewan yang dilindungi karena populasinya memang sedikit. Komodo hanya dapat ditemukan di Indonesia, tepatnya di Pulau Komodo. Namun, hewan ini juga mulai menjadi langka karena ulah beberapa oknum. Habitat komodo semakin menyusut dan hampir punah.

Badak Bercula Satu

Badak bercula satu hanya dapat ditemukan di Pulau Jawa, tepatnya di wilayah Ujung Kulon. Hewan ini masuk ke daftar hewan yang dilindungi di Indonesia karena jumlahnya yang hanya tersisa 50 ekor saja.

Perbedaan class dan id pada css

Class dan id dalam css digunakan sebagai penanda atau pemilih terhadap element html mana saja stylingnya akan berlaku. Namun diantara keduanya memiliki bebrapa perbedaan, antara lain:

  • Di dalam css class di didefinisikan dengan menggunakan tanda titik “.”, sedangkan id dengan tanda pagar (“#”)
  • Di dalam sebuah dokumen html, setiap class dapat diberikan kepada banyak element, sedangkan untuk id hanya bisa di berikan kepada satu element saja untuk masing-masing id
  • Di dalam sebuah dokumen html, sebuah element html bisa memiliki beberapa class sekaligus, sedangkan untuk id satu element hanya bisa memiliki satu id

Dengan demikinan dapat disimpulkan bahwa dalam konteks sebuah dokumen html, class bersifat umum, artinya bisa dipakai secara berulang-ulang di dalam sebuah dokumen yang sama. Sedangkan id bersifat unik yang mana hanya bisa dipakai sekali saja yaitu untuk element html tertentu.
